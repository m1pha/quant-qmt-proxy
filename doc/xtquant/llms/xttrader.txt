## 一、模块概述

`xttrader` 是 `xtquant` 库中负责交易操作的核心模块，支持：
- 同步/异步下单与撤单
- 资产、持仓、委托、成交查询
- 回调机制（订单状态、成交推送等）
- 信用交易、新股申购、资金划拨等高级功能

依赖 MiniQMT 客户端，确保交易安全与一致性。

## 二、核心概念

- **XtQuantTrader**: 主交易类，需传入 `path`（MiniQMT 数据路径）和 `session_id`（会话ID，避免冲突）
- **StockAccount**: 证券账号类，构造如 `StockAccount('1000000365', 'STOCK')`
- **回调机制**: 通过继承 `XtQuantTraderCallback` 实现事件处理
- **委托类型**: `'LO'`（限价单）、`'MO'`（市价单）、`'AO'`（竞价单）等
- **方向**: `'buy'`、`'sell'`
- **时间有效性**: `'Day'`（当日有效）、`'GTC'`（撤销前有效）

## 三、初始化与连接

```python
from xtquant.xttrader import XtQuantTrader, XtQuantTraderCallback
from xtquant.xttype import StockAccount

class MyCallback(XtQuantTraderCallback):
    def on_disconnected(self):
        print('连接断开')
    def on_stock_order(self, order):
        print(f'订单: {order.stock_code} {order.order_status}')
    def on_stock_trade(self, trade):
        print(f'成交: {trade.stock_code} {trade.volume}@{trade.price}')
    def on_stock_asset(self, asset):
        print(f'资产: {asset.cash} {asset.total_asset}')
    def on_stock_position(self, position):
        print(f'持仓: {position.stock_code} {position.volume}')
    def on_order_error(self, order_error):
        print(f'下单错误: {order_error.error_msg}')
    def on_cancel_error(self, cancel_error):
        print(f'撤单错误: {cancel_error.error_msg}')

# 初始化
path = r'D:\迅投极速交易终端 睿智融科版\userdata_mini'
session_id = 123456  # 每个策略用唯一ID
xt_trader = XtQuantTrader(path, session_id)
xt_trader.register_callback(MyCallback())
xt_trader.start()
connect_result = xt_trader.connect()
print('连接结果:', connect_result)

# 订阅账号
acc = StockAccount('1000000365')
subscribe_result = xt_trader.subscribe(acc)
print('订阅结果:', subscribe_result)
```

## 四、下单与撤单

### 4.1 同步下单

```python
# 股票限价买入
order_result = xt_trader.order_stock_sync(
    account_id='1000000365',
    stock_code='000001.SZ',
    side='buy',
    quantity=100,
    price=12.34,
    order_type='LO',
    time_in_force='Day'
)
print('下单结果:', order_result)  # 返回订单ID或错误信息
```

### 4.2 异步下单

```python
# 异步下单，需实现 on_order_stock_async_response 回调
async_result = xt_trader.order_stock_async(
    account_id='1000000365',
    stock_code='000001.SZ',
    side='sell',
    quantity=50,
    price=13.00,
    order_type='LO'
)
print('异步下单提交:', async_result)
```

### 4.3 撤单

```python
# 同步撤单
cancel_result = xt_trader.cancel_stock_order(order_id='订单号')
print('撤单结果:', cancel_result)

# 异步撤单
cancel_async = xt_trader.cancel_stock_order_async(order_id='订单号')
```

### 4.4 批量下单

```python
# 批量同步下单
orders = [
    {'stock_code': '000001.SZ', 'side': 'buy', 'quantity': 100, 'price': 12.34},
    {'stock_code': '600000.SH', 'side': 'sell', 'quantity': 200, 'price': 8.00}
]
batch_result = xt_trader.order_stock_batch_sync(account_id='1000000365', orders=orders, order_type='LO')
```

## 七、使用建议

- **会话管理**: 每个策略使用唯一 `session_id`，避免冲突。
- **回调处理**: 实现必要的回调方法，处理订单状态变化。
- **错误处理**: 检查下单/撤单返回值，处理 `on_order_error` 和 `on_cancel_error`。
- **性能**: 异步接口适用于高频交易，减少阻塞。
- **安全**: 敏感操作如资金划拨需谨慎，测试环境验证。
- **连接维护**: 监听 `on_disconnected`，必要时重连。

## 八、示例：简单交易策略

```python
from xtquant.xttrader import XtQuantTrader, XtQuantTraderCallback
from xtquant.xttype import StockAccount
import time

class TradingCallback(XtQuantTraderCallback):
    def __init__(self, trader):
        self.trader = trader
        self.positions = {}

    def on_stock_position(self, position):
        self.positions[position.stock_code] = position.volume

    def on_stock_trade(self, trade):
        print(f'成交: {trade.stock_code} {trade.volume}股 @{trade.price}')

# 初始化
path = r'D:\迅投极速交易终端 睿智融科版\userdata_mini'
trader = XtQuantTrader(path, session_id=1001)
callback = TradingCallback(trader)
trader.register_callback(callback)
trader.start()
trader.connect()
acc = StockAccount('1000000365')
trader.subscribe(acc)

# 简单策略：买入特定股票
if '000001.SZ' not in callback.positions or callback.positions['000001.SZ'] < 100:
    trader.order_stock_sync(
        account_id='1000000365',
        stock_code='000001.SZ',
        side='buy',
        quantity=100,
        price=12.00,  # 假设价格
        order_type='LO'
    )

# 运行循环
while True:
    time.sleep(10)
    # 检查持仓、资产等
    asset = trader.query_asset('1000000365')
    print(f'当前现金: {asset.cash}')
```
